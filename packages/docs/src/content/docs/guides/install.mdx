Pasos para instalar `@functional-ts/monads` en tu proyecto:

1. Instalar la librería con npm, pnpm o yarn:

<Tabs>
  <TabItem label="npm">
    ```bash
    npm install @functional-ts/monads@latest
    ```
  </TabItem>
  <TabItem label="pnpm">
    ```bash
    pnpm add @functional-ts/monads@latest
    ```
  </TabItem>
  <TabItem label="yarn">
    ```bash
    yarn add @functional-ts/monads@latest
    ```
  </TabItem>
</Tabs>

2. Importar la librería en tu código:

```typescript
import { maybe } from '@functional-ts/monads';
```

3. ¡Listo! Ya puedes empezar a usar la librería en tu proyecto.
